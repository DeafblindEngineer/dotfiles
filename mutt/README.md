### Mutt Configuration

### Initial Setup

```
$ sudo apt install mutt
$ mkdir -p ~/.mutt/cache/headers
$ mkdir ~/.mutt/cache/bodies
$ touch ~/.mutt/certificates
$ touch ~/.mutt/muttrc
$ vim ~/.mutt/muttrc
```

$ vim ~/.mutt/imap01
